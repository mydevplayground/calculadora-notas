# Calculadora de Média de Notas

Este projeto é uma pequena aplicação em JavaScript para registrar notas, exibir uma lista com os valores inseridos, calcular a média e mostrar o status final do aluno de acordo com o resultado obtido.

O objetivo deste repositório é documentar estudos e prática de lógica de programação e manipulação do DOM.

## Funcionalidades

* Inserção de notas numéricas
* Exibição das notas em uma lista
* Cálculo automático da média
* Exibição do status final (Aprovado, Recuperação ou Exame Final)
* Função para limpar notas e reiniciar o cálculo

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Insira as notas e utilize os botões disponíveis na interface

## Finalidade
Foi desenvolvido exclusivamente para fins educativos, servindo como registro de prática em JavaScript.
